Project Fugu is an initiative by companies like Google, Intel and Microsoft that aims to extend the web with native-like capabilities like ability to access hardware APIs and storage in a safe manner that can be easily understood by users. Join us as Kenneth, co-inventor of PWA, give us an introduction about Project Fugu and give the latest and upcoming web capabilities we can take advantage of in our Progressive Web Apps.

We will be streaming live via Facebook, on YouTube and Twitter and will be available on demand via PWA Pilipinas App.

The event is FREE and is open to the public but to know our audience better, please register with your details and possibly leave a question so that we can ask it during the livestream.

All speakers, attendees and volunteers are required to observe and follow the PWA Pilipinas Code of Conduct that you can read here.

About Kenneth Christiansen

Kenneth is a Danish software engineer and architect, specialized in mobile and web technology. Working at Intel out of Copenhagen, Kenneth helps define Intel's strategy and plans regarding the Web platform, as well as put it in action. Before joining Intel, Kenneth was employed by Nokia (Denmark and Brazil) where he worked on the Nokia N9 web browser, Qt, WebKit, as well as many other mobile projects.

Kenneth is an elected member of the W3C Technical Architect Group (TAG). Currently Working on making Progressive Web Apps more capable as part of Project Fugu.
